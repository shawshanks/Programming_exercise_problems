## 寻找序列中某项的索引 p20
```python
# 移动一个字符串中的索引直到发现值为`'x'`的项，或者到达序列末尾
# Advance an index through a sequence of characters until finding an entry with value 'X' 
# or reaching the end of the sequence 

j = 0
while j < len(data) and data[j] = 'X':
  j += 1
  
for j in len(data):
  if data[j] == 'X':
     return j

```
